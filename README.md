Fuel-ansible
============

Scratchbook and utilities to operate on MOS clusters using Ansible

Installing Ansible on the master node (8.0)
===========================================

Install gcc, python-pip, python-virtualenv, python-crypto
Virtualenv && pip install -e
Autoremove gcc
